# CDKK - Community Download & Sharing Network

Welcome to the CDKK website! This project serves as a community hub for users to download, share, and connect with each other. Below are the details regarding the structure and features of the website.

## Project Structure

```
cdkk-website
├── index.html        # Main entry point of the CDKK website
├── y2k.html          # HTML page designed with a 2000s aesthetic
├── downloads.html     # Page listing downloadable content organized by categories
├── users.html        # Page providing information about users in the community
├── blog.html         # Blog section featuring articles and updates
├── forum.html        # Community discussion page for user engagement
├── Assets            # Directory containing all assets used in the website
│   ├── Background     # Background images used throughout the website
│   ├── Favicon        # Favicon files for the website
│   ├── Fonts          # Font files used in the website
│   └── Image          # Image files used in the website
└── README.md         # Documentation for the project
```

## Features

- **User-Friendly Navigation**: Easy access to all sections of the website.
- **Community Focused**: Designed to foster connections and sharing among users.
- **Downloadable Content**: A wide range of files available for download.
- **Blog and Forum**: Sections dedicated to community news and discussions.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open `index.html` in a web browser to view the homepage.
3. Navigate through the website using the provided links.
